## Module Gating Implementation
1. On mount, `loyalty.getConfig()` is called
2. If `config.enabled === false`, shows "Enable Loyalty Module" placeholder UI
3. On enable click, calls `loyalty.updateConfig({ enabled: true })` then reloads
4. If enabled, shows full configuration UI:
   - Points Configuration section (accumulation mode, rates, expiration)
   - Tier Definitions DataTable with edit buttons
   - Rewards Catalog DataTable with edit buttons

- Rewards table: name, points cost, description, edit action
