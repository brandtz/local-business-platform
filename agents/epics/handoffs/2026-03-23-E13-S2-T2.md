## SDK Methods Consumed

## Features
1. **Search Toolbar** — text search input with 300ms debounce
2. **Category Filter** — dropdown select from loaded categories
3. **Sort Options** — Name (A-Z/Z-A), Price (Low-High/High-Low)
4. **Active Filter Tags** — removable tags for active search and category filters
5. **Product Grid** — card grid with name, description, price, compare-at price
6. **Pagination** — page navigation with Previous/Next and numbered pages

## Query Parameters
- `?category=<categoryId>` — pre-selects category filter from URL

## Loading/Error/Empty State Handling
- Loading: spinner with "Loading catalog..." message
- Error: alert with "Failed to load products" message
- Empty: "No products found — try adjusting your search or filters"
