# E13-S6-T5 Handoff: Content Pages Editor (BA-14)

## Date: 2026-03-23

## Component Composition
- `ContentPagesPage` (pages/content-pages-page.ts)
- Uses: TabBar (content tabs), CardGrid (page cards with status badges), Full-page editor (title, slug, body, SEO), RichTextEditor (textarea implementation)
- View-model helpers in `content-views.ts`

## Editor Form Fields
### Content Section
| Field | Type | Required | Notes |
|-------|------|----------|-------|
| title | text | Yes | Page title (large input) |
| slug | text/prefix | No | URL slug with "/" prefix, auto-generated from title |
| body | textarea | No | Rich text editor (textarea for now) |

### SEO Section
| Field | Type | Required | Notes |
|-------|------|----------|-------|
| seoTitle | text | No | Meta title |
| seoDescription | textarea | No | Meta description |
| featuredImage | text | No | Featured image URL (ogImageUrl) |

## Draft/Publish Workflow
- "Save Draft" button → saves with `draft` status
- "Publish" button → saves with `published` status
- Editor replaces list view when open

## CardGrid Display
- Title with status badge (Draft/Published/Archived)
- URL slug display
- Excerpt from body or SEO description
- Updated date
- Edit and Delete action buttons

## SDK Methods Consumed

| `content.updatePage()` | `string, UpdateContentPageRequest` | `ContentPageRecord` |
| `content.deletePage()` | `string` | `void` |
