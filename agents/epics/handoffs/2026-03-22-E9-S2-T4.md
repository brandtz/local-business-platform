## Admin API Endpoints (Response Shapes)

### Upload Artifact
```typescript
POST /api/admin/tenants/:tenantId/imports/artifacts
Response: { success: boolean; artifact: ImportArtifactRecord; job: ImportJobRecord }
```

### List Artifacts
```typescript
GET /api/admin/tenants/:tenantId/imports/artifacts?artifactType=&uploadStatus=&page=&pageSize=
Response: { items: ImportArtifactRecord[]; total: number; page: number; pageSize: number; totalPages: number }
```

### List Jobs
```typescript
GET /api/admin/tenants/:tenantId/imports/jobs?status=&page=&pageSize=
Response: { items: ImportJobRecord[]; total: number; page: number; pageSize: number; totalPages: number }
```

### Retry Job
```typescript
POST /api/admin/tenants/:tenantId/imports/jobs/:jobId/retry
Response: { success: boolean; jobId: string; newStatus: ImportJobStatus; reason?: string }
```
