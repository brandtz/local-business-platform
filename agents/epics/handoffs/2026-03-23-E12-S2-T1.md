## Entitlement Consumption Contract

Subscription entitlements are the **authoritative source** of feature availability:
1. If tenant has an active subscription → subscription entitlements determine modules
2. If no subscription → manual module assignments (E3-S5) are used as fallback
3. Source field in result: `"subscription"` | `"manual"` | `"merged"`
4. Restricted modules: manual modules not in subscription entitlements
