|-------|------|----------|-------|
| name | text | Yes | Location name |

| state | text | Yes | State |
| zip | text | Yes | ZIP code |

| email | email | No | Email address |

- 7-day grid with Mon-Sun
- Each day: closed checkbox, open time, close time
- Default: weekdays 9AM-5PM, weekends closed
