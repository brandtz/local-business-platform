## Component Composition
- `CatalogProductsPage` (pages/catalog-products-page.ts)
- Uses: TabBar (catalog tabs), DataTable (product list), SearchToolbar (search + filters), SlidePanel (create/edit form), BulkActions bar, Modal (bulk confirm)
- View-model helpers in `catalog-views.ts`

## SlidePanel Form Fields
### Basic Info Section
| Field | Type | Required | Notes |
|-------|------|----------|-------|
| name | text | Yes | Product name |
| slug | text | No | Auto-generated from name |
| description | textarea | No | Product description |
| categoryId | select | No | Category dropdown |

### Pricing Section
| Field | Type | Required | Notes |
|-------|------|----------|-------|
| price | number | Yes | Price in cents |
| compareAtPrice | number | No | Compare-at price in cents |

## SearchToolbar Filters
- Text search by product name
- Category dropdown filter
- Status dropdown filter (Active/Inactive)

## Bulk Actions Implementation
- Checkbox selection per row + "select all" header checkbox
- Bulk action bar appears when items selected
- Actions: Activate, Deactivate, Delete
- Confirmation modal before executing bulk operations
- Sequential API calls per selected item
