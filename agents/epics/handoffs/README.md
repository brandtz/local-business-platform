# Handoff Log

This directory stores one handoff note per completed, partially completed, or blocked technical task.

## Naming Pattern

Use:
- YYYY-MM-DD-task-id.md

Example:
- 2026-03-15-E1-S1-T1.md

## Rules

- one note per task state handoff
- if a task changes from blocked to completed, create a new note rather than rewriting history
- each note should reference the immediately previous dependency note when relevant
- notes should summarize behavior, contracts, tests, and next tasks, not every file edit
