## Calendar Event Positioning Algorithm
1. **Date Range Calculation**: Based on view type (day/week/month), compute start and end dates
   - Day: same date for start/end
   - Week: Sunday to Saturday of the current week
   - Month: first to last day of the current month
2. **Day Structure**: For each day in range, create a `CalendarDayViewModel` with date, label, and booking blocks
3. **Block Mapping**: Each booking is mapped to a `CalendarBlockViewModel` with status badge, times, staff, customer
4. **Day View**: 30-minute time slot grid with blocks placed at matching start times
5. **Week View**: Day columns with stacked booking cards per day
6. **Month View**: Day cells showing booking count badges; click to drill into day view

## Implementation Details
- View toggle: Day | Week | Month
- Date navigation: Previous, Today, Next buttons
- Booking cards display: service, customer, time range, staff, status color
