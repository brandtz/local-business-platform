## Order State Machine

- States: draft → placed → confirmed → preparing → ready → completed
- Cancellation: allowed from placed, confirmed, preparing only
- Terminal states: completed, cancelled (no outgoing transitions)
- Fulfillment modes: pickup, delivery, dine-in
