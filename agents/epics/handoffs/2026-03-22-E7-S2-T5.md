## Time-Ago Display

### formatTimeAgo(timestamp, now?)
- Converts ISO 8601 timestamp to relative time string
- Intervals: "just now" (<60s), "1 min ago", "N mins ago", "1 hour ago", "N hours ago", "1 day ago", "N days ago", "1 month ago", "N months ago", "1 year ago", "N years ago"
- Injectable `now` parameter for deterministic testing
- Future timestamps return "just now"

### getOrderDisplayTimestamp(order)
- Returns the most relevant timestamp for display in admin pipeline views
- Priority: cancelledAt > completedAt > readyAt > preparingAt > confirmedAt > placedAt > createdAt
- Always returns a valid timestamp (falls back to createdAt)
