## Customer Booking Tracking Read Model

### BookingTrackingViewModel
```typescript
{
  bookingId: string;
  isCancelled: boolean;
  isNoShow: boolean;
  progressBar: BookingProgressBarStep[];  // 4 steps
  progressPercent: number;  // 0-100
  statusMessage: string;    // Human-readable status
  serviceInfo: BookingServiceInfoViewModel;
}
```

### Progress Bar Steps
1. **Requested** → "Your booking has been requested and is waiting to be confirmed."
2. **Confirmed** → "Your booking has been confirmed. See you soon!"
3. **Checked In** → "You've checked in. Your appointment is in progress."
4. **Completed** → "Your appointment has been completed. Thank you!"

Cancelled: "Your booking has been cancelled."
No-Show: "This booking was marked as a no-show."

### Status-Bar Progress Mapping
- Each step has state: completed | current | upcoming | skipped
- Cancelled/no-show bookings show completed steps with timestamps, remaining as skipped
- Progress percent = (completed + current steps) / total steps × 100
