## Tenant Isolation Guarantee
- SearchIndexRepository uses a `Map<tenantId, Map<entityType, Map<id, doc>>>` structure
- Every query path requires tenantId as the first lookup key
- No API exists to query across tenants
- 9 repository tests verify tenant isolation behavior
