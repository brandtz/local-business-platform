## Query API Endpoints

All endpoints require tenantId + time filter (from, to). Optional: locationId, periodType.

1. **KPI Summary** — getKpiSummary(params) → AnalyticsKpiSummary
   - Returns current/previous values + changePercent + trend direction for: revenueCents, orderCount, bookingCount, newCustomerCount, returningCustomerCount, retentionRate
   - Previous period auto-calculated from time filter duration

2. **Revenue Time Series** — getRevenueTimeSeries(params, periodType?) → RevenueTimeSeriesResponse
   - Returns time series data points + total revenue

3. **Volume Time Series** — getVolumeTimeSeries(params, periodType?) → VolumeTimeSeriesResponse
   - Returns separate order and booking volume points

4. **Channel Breakdown** — getChannelBreakdown(params) → ChannelBreakdownResponse
   - Returns per-channel counts, revenue, and revenue percentage

5. **Top Performers** — getTopPerformers(params, category, limit?) → TopPerformersResponse
   - Categories: product, staff, location. Default limit 10.

6. **Retention Insights** — getRetentionInsights(params, periodType?) → RetentionInsightsResponse
   - Returns retention/churn rates + time series
