**Task ID:** E11-S2-T7

## Changed Surfaces

### Types Package (`packages/types/src/loyalty.ts`)

**Loyalty Tab Data Contract (CP-15):**
- `LoyaltyTabData`: tierName, pointBalance, lifetimePoints, nextTierName, nextTierThreshold, progressPercent, pointsToNextTier, memberSince, canRedeem, minimumRedemptionPoints, recentActivity[]
- `buildLoyaltyTabData()`: computes tab data from account + config — calculates progress bar percentage, next-tier info, redeem eligibility

**Admin Customer Tags:**
- `CustomerTagType`: `vip` | `loyalty` | `new`
- `CustomerTag`: type + label
- `AdminCustomerLoyaltySummary`: customerId, tenantId, currentTier, pointBalance, lifetimePoints, tags[]
- `deriveCustomerTags()`: derives VIP (gold/platinum), Loyalty (lifetime > 0), New (member < 30 days)

### API Service (`apps/api/src/loyalty/loyalty.service.ts`)
- `getLoyaltyTabData(tenantId, customerId)`: builds full tab data with recent activity (last 5 entries)
- `getAdminCustomerLoyaltySummary(tenantId, customerId)`: returns summary with derived tags
- `getAdminCustomerLoyaltyList(tenantId)`: lists all customers with loyalty summaries
