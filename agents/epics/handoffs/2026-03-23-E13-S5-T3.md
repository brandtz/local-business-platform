# Handoff: E13-S5-T3 — Business Profile and Branding (BA-02)

## Task

Build the business profile form and branding configuration page with a live preview panel showing header with primary color and business name.

## Admin Route Tree with Auth Guards and Module Gating

The profile/branding page is mounted at `/admin/profile` inside the AdminLayout. The route requires authentication but is not gated by a specific module — every tenant has access to their own profile settings.

```
/admin/profile   (meta.requiresAuth: true, gatingKey: "core")
```

## Sidebar Navigation Data Structure with Gating Keys

```ts
{
  label: "Profile & Branding",
  icon: "building",
  route: "/admin/profile",
  gatingKey: "core"
}
```

## Chart Library Choice and Usage

No chart library is needed for this page.

## SDK Methods Consumed

- `tenants.get()` — fetches the current tenant profile and branding configuration.
- `tenants.update()` — persists profile edits and branding changes (logo URL, colors, font selections).

## Branding Preview Implementation Approach

The page includes a **live preview panel** rendered alongside the branding form. Implementation details:

1. **Reactive binding** — The preview panel reads directly from the form's reactive state, so every keystroke or color picker change is reflected instantly.
2. **Header preview** — A mock header element applies the selected `primaryColor` as its `background-color` and renders the `businessName` text.
3. **Logo preview** — When a logo file is selected in the upload area, a local object URL is created and displayed in the preview header. If no logo is set, a placeholder icon is shown.
4. **Color pickers** — Standard `<input type="color">` elements bound to `TenantBrandConfig.primaryColor` and `TenantBrandConfig.accentColor`.
5. **Validation** — Profile fields are validated via `validateTenantProfile` from `tenant-profile.ts` before submission.

```ts
interface TenantBrandConfig {
  primaryColor: string;
  accentColor: string;
  logoUrl: string | null;
  fontFamily: string;
}
```
