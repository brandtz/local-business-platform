## Changed Surfaces

### Types Package (`packages/types/src/search.ts`)
- `AutocompleteQuery` — tenantId, prefix, optional entityTypes[], optional limit
- `AutocompleteResult` — suggestions[]
- `AutocompleteSuggestion` — text, entityType, entityId, score
- Constants: `AUTOCOMPLETE_DEFAULT_LIMIT` (5), `AUTOCOMPLETE_MAX_LIMIT` (20), `AUTOCOMPLETE_MIN_PREFIX_LENGTH` (1)
- Validation: `validateAutocompleteQuery()`

### API Package (`apps/api/src/search/search.service.ts`)
- `SearchService.autocomplete()` — prefix-based typeahead from indexed documents
  - Tenant-scoped: only returns suggestions from the querying tenant
  - Case-insensitive prefix matching on searchableText fields
  - Entity type filtering (optional)
  - Score-based ranking (exact match > prefix match ratio)
  - Alphabetical tiebreaking
  - Limit enforcement (max 20)
