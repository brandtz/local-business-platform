## Filter-Toolbar Metadata Contract
Frontend consumes `FilterToolbarDefinition` to render filter controls generically:
- `type: "enum"` → dropdown/multi-select with `options[]`
- `type: "range"` → number inputs with `rangeMin`/`rangeMax`
- `type: "date"` → date picker pair
- `type: "text"` → text input with `placeholder`
- `type: "boolean"` → toggle switch

Each toolbar includes `defaultSort` and `sortOptions[]` for column header sorting.
