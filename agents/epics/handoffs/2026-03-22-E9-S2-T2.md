## Import Job Lifecycle States
```
pending → processing → staged → completed
    ↘          ↘         ↘
     failed ← ─ ─ ─ ─ ─ ─
       ↓
    pending (retry)
```

## Artifact Storage Contract
- Tenant-scoped paths: `{tenantId}/imports/{artifactId}/{sanitizedFilename}`
- Malware scan hook: `MalwareScanner.scan()` called before storage commit
- Fail-safe: `NoOpMalwareScanner` skips scan when no scanner configured
- Scan failures → artifact rejected, no job created

## Admin API Endpoints
