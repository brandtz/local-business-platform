## Component Composition

- Uses: TabBar (content tabs), DataTable (announcement list), SlidePanel (create/edit form), Activate/Deactivate toggle

| Field | Type | Required | Notes |
|-------|------|----------|-------|
| placement | select | No | Banner / Popup / In-feed |
| isActive | checkbox | No | Active toggle |

### Scheduling Section
| Field | Type | Required | Notes |
|-------|------|----------|-------|
| startDate | datetime-local | No | Start date/time |
| endDate | datetime-local | No | End date/time |

## Status Logic
- `getAnnouncementStatusLabel()` derives status from isActive + dates:
  - Inactive: isActive is false
  - Scheduled: isActive but startDate is in the future
  - Expired: isActive but endDate is in the past
  - Active: isActive and within date range (or no dates)

## Activate/Deactivate Toggle
- Inline button in table actions column
- Calls `content.updateAnnouncement(id, { isActive })` directly
- Refreshes list after toggle
