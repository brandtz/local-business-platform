## CRM Slide Panel Lazy Loading Approach
1. Customer list loads on page mount via `customers.list()`
2. No customer detail data is preloaded — only list rows are rendered
3. On row click, `loadCustomerDetail(customerId)` is called
4. Panel opens with loading state while two parallel requests fire:
   - `customers.get(customerId)` for profile
   - `customers.getMetrics()` for aggregate metrics (gracefully handles failure)
5. Panel renders contact info, metrics grid once data arrives
6. Panel closes without affecting list state
