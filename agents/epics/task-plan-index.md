## Minimal Context Pack for a New Agent

A new agent assigned to a task should usually read only:
- the relevant epic story file
- the relevant epic task plan file
- ../PLAYWRIGHT_AGENT_STANDARD.md when the task affects browser-visible behavior
- technical-tasking-standard.md
- test-requirements-standard.md
- task-ownership-model.md
- agent-handoff-standard.md
- handoff-note-template.md
- the latest handoff note from the immediately preceding dependency

If more context is needed, the upstream handoff was incomplete or the task is too broad.
